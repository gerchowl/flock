# Debt census

Structured, gate-derived views of pre-existing debt we're paying down
incrementally. Each section is regenerated by its declared command
(`guardrails-derived-docs --fix docs/DEBT.md`); do not hand-edit the region.

## `no-raw-trace-fields` — sites still marked for migration to the `src/logging.rs` facade

The `no-raw-trace-fields` gate confines raw Debug/Display field formatters
(`?expr` / `%expr`) to the schema surface (`src/logging.rs`). Every other
existing site carries a `guardrails-ok(no-raw-trace-fields)` marker (set by
`scripts/mark_trace_fields.py`) so the gate can go hard immediately and police
new code while the logging-redesign phase migrates the sites into named facade
functions. This census tracks per-file marker counts so PRs can burn it down
visibly.
